# Tooling — people-context

Framework and runtime guides for the people-context service.

- **Bun** — https://bun.sh/docs — TypeScript runtime, test runner, bundler.
- **Elysia** — https://elysiajs.com — Type-safe HTTP framework for Bun.
- **postgres.js** — https://github.com/porsager/postgres — PostgreSQL driver.
- **nats.js** — https://github.com/nats-io/nats.js — NATS client with JetStream.
